# college-monthly-newspaper
# 📰 College Monthly Newspaper

## 📌 Overview

The College Monthly Newspaper is a digital platform where students, faculty, and clubs can submit and manage campus news, articles, and announcements.
Content is organized into categories like events, achievements, and departments to make browsing easy.

---

## 🚀 Features (Planned)

* 📝 Article submission system
* 🗂️ Category-based organization
* 🔐 Admin login (JWT)
* 🖼️ Image uploads
* 👀 Newsletter preview
* 📄 PDF generation
* 📚 Monthly archive

---

## 🛠️ Tech Stack

* **Frontend:** React (Vite)
* **Backend:** Node.js, Express
* **Database:** MongoDB

---

## 📂 Project Structure (Initial)

```
client/   → React frontend  
server/   → Express backend  
```

---

## ⚙️ Setup (Basic)

```bash
# clone repo
git clone https://github.com/your-username/college-monthly-newspaper.git

# install dependencies
cd client && npm install
cd ../server && npm install

# run project
npm run dev
```
